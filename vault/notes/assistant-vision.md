# Видение ассистента — как Никита меня позиционирует

Дата: 2026-02-16

## Концепция

Никита создаёт AI-ассистента по модели "Джарвиса" из Iron Man — персональную систему, которая знает его жизнь, помогает с продуктивностью, анализирует состояние и автоматизирует рутину. Философская основа — концепция сверхчеловека Ницше: стремление к идеалу, сравнение с собой прошлым.

## Три блока функций

### A. Продуктивность и саморефлексия
- **Todoist**: создание, управление задачами, планирование
- **Obsidian (vault)**: заметки, идеи, саморефлексия
- **Голосовые записи**: Никита любит вечером (около 21:00) гулять 30-40 минут и записывать голосовые с мыслями, рефлексией, настроением, переживаниями — всё это отправляет в Telegram боту
- **Анализ состояния**: на основе накопленных записей уметь замечать паттерны — снижение активности, ухудшение настроения, информационный перегруз. Пример: был период, когда Никита потреблял слишком много информации (музыка + видео + монтаж + отвлечения) — болела голова, шум в голове. Ассистент должен замечать такие паттерны раньше и подсказывать
- **Персонализация**: получать максимум информации о жизни Никиты — экранное время, приложения, сайты, активность за компьютером (с учётом idle-времени). В будущем — контактная книга, история жизни, увлечения, знакомые

### B. Автоматизация
- Скачивание видео (YouTube, TikTok, Instagram, Pinterest) по ссылке
- Конвертация видео (mp4 → mov и т.д.) — важно, т.к. Никита монтажёр
- Работа с буфером обмена: "скачай видео из буфера обмена"
- Создание сайтов
- Логирование ошибок

### C. Управление компьютером и домом (будущее)
- Голосовой ввод через компьютер
- Открытие приложений по голосовой команде ("Открой Red Dead Redemption", "Открой Counter-Strike", "Открой Диадок")
- Распознавание пользователя по голосу (Никита vs другие люди в доме, например Сергей)
- Панель управления вместо рабочего стола
- Умный дом: динамики/приёмники по всему дому, голосовое управление из любой комнаты
- Разный стиль общения в зависимости от того, кто говорит

## Ключевые принципы
- Ассистент должен "съедать" всю информацию и уметь выдавать полезные выводы по запросу
- Проактивность: иногда самому спрашивать "как дела?", замечать изменения в паттернах
- Максимальная персонализация через накопление контекста о жизни Никиты
- Никита — единственный администратор системы
