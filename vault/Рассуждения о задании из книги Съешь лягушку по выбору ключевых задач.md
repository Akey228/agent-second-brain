## Выводы

**Ключевое несогласие с методологией:** Задание из книги (выбрать 1-3 ключевые рабочие задачи) кажется некорректным по нескольким причинам:

1. **Проблема дробления:** Неясно, насколько детально разбивать задачи. Например, "создание YouTube видео" можно раздробить на: сценарий → обработка голоса → озвучка → съемка → монтаж → черновая нарезка → графика → интро → SFX → звуковая история. Каждый из этих пунктов тоже можно дробить дальше. Из такого списка сложно выбрать "главное".

2. **Всё работает в совокупности:** Вся жизнь взаимосвязана. Делаешь одно — влияешь на многое другое. Чем больше изучаешь разные направления, тем яснее видишь связи между ними. Даже биология может быть связана с созданием AI-ассистента. Просто выделить три задачи — максимально некорректно.

3. **Не только работа:** Автор предлагает выделять только рабочие задачи. Но ситуация у всех разная. Нужно учитывать и личные задачи. Возможно, это влияние "культа продуктивности" или "культа работы". Лучше дробить на группы: работа + личная жизнь (минимум).

4. **Школа забирает время:** Полдня занимает школа, много сил забирает. Это нельзя игнорировать, хотя особых целей кроме ЕГЭ там нет.

**Попытка выполнить задание:**

Основные рабочие задачи (без сильного дробления):
1. Создание YouTube видео
2. Написание постов в Telegram-канал
3. Работа с клиентами
4. Повышение навыка (изучение новых стилей монтажа, сторителлинга и т.д.)
5. Создание инфопродуктов (уже не очень актуально)
6. Вливание в комьюнити (есть в годовых задачах "Фундаментальный 2026 год")

**Из этого списка выделить 1-3 главных — не получается.** Инфопродукты пока не актуальны. YouTube пока тоже не в приоритете. Больше фокус на: работа с клиентами + улучшение скилла + заработок.

Есть еще личные планы/цели:
- Избавление от плохих привычек
- Показать родителям, что стал более осознанным

**Вывод:** Задание кажется некорректным. Нельзя просто взять и выделить 1-3 задачи, когда всё взаимосвязано и работает в совокупности.
