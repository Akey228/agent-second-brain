В видосике автор здрво обхяснил как шустро делать форматирование заметки

1. Для этого открываем Vs Code 
2. Открываем любую встроенную нейронку, в видео была показана Cody
3. Отмечаем эту заметку как файл
4. И делаем промт "сделай красиво"
5.  Дальше нажимаем Apply, после чего файл меняется и приобретает красивое, понятное форматирование
6. Это сразу же показывается в обсидиан
