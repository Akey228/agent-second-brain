При создании видео часто используется множество различных файлов: сценарий, план действий, список дел и другие. Было бы удобно иметь возможность добавлять специальный файл или несколько файлов для трекинга процесса создания видео. Этот трекер мог бы отслеживать все этапы работы, позволяя просто ставить галочки на каждом шаге.

Процесс можно делить на несколько основных этапов: сначала появляется идея, её нужно тщательно обдумать и проработать. Затем создаётся структура папок для проекта в Обсидиане, которую также можно отслеживать в трекере. Наконец, завершающий этап — публикация видео и его распространение по группам и другим каналам. Каждый из этих шагов сопровождается отметками в виде галочек.

Такой трекер можно встроить либо в основной файл проекта (мок файл), связанный с конкретным видео, либо реализовать в виде отдельного списка дел. Это поможет организовать рабочий процесс, видеть прогресс и не упускать важные этапы создания и публикации видео.
