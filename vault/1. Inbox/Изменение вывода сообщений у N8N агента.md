Для N8N агента необходимо изменить вывод сообщений. Теперь в личные сообщения будет отправляться только подтверждение о успешном создании заметки. В сообщении нужно указывать только название заметки из Obsidian, без дополнительного текста или JSON файлов. Формат сообщения должен быть: "Заметка, название заметки, успешно создано."
