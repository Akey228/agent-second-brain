Агент должен иметь доступ к факторным базам знаний, содержащим расходы и доходы, таблицы мира с маркетингом, файлы из [[MOC - Obsidian]] и Яндекс.Диска (через API для синхронизации). Внутри агента должны быть разные специалисты: маркетолог, специалист по продажам, монтажник и т.д. Все файлы, связанные с работой, должны быть подключены.
